# 项目名称

链上人格测试/Solana钱包MBTI测试

## 💻 项目 Repo

https://github.com/z1gui/wallet-personality


## 📌 项目简介

这不是玄学，这是链上行为画像。
连接你的 Phantom 钱包，系统会用 Helius API 读取公开交易记录，产出一个“毒舌但合理”的 MBTI 性格标签。

## 🛠️ 技术栈


- 前端：Vite
- 后端：node + Express
- 工具：Helius API
